## Project Overview
This is a **fully interactive test online shop** developed as part of an advanced JavaScript module. The primary objective was to create a **scalable and maintainable product management interface** with real-world functionalities using **HTML, CSS, JavaScript**, and modern front-end libraries. The project demonstrates a structured approach to **dynamic UI rendering, state management, and user interactions**, adhering to best practices for clean, readable, and maintainable code.

Key functionalities include:

- Dynamic addition, update, and deletion of products
- Real-time search and filtering of product lists
- Sorting and discount application
- Shopping cart management with persistent state via **localStorage**
- Interactive user notifications via **SweetAlert2**
- Fully responsive layout using **Bootstrap 5**

- Interface updates automatically without page reloads  

### Shopping Cart
- Add and remove products from the cart  
- Real-time cart update and management

- **Real-time search** using List.js  
- Product sorting implemented in **pure JavaScript**  
- Supports filtering and dynamic rendering

- Mobile-first approach with **Bootstrap 5 grid system**  
- Adapts seamlessly across different screen sizes
